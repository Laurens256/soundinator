# 👋🏼 Soundinator

---

## 👩🏼‍💻 How to use
1. Clone this repository
```
$ git clone https://github.com/Laurens256/soundinator.git
```

2. Install dependencies
```
$ cd soundinator
$ npm i
```

3. Run the server
```
$ npm run dev
```
In your browser, go to `http://localhost:8080`
